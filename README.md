# ⚙️ Sales Automation Dashboard (Excel VBA)

## 📌 Project Overview
This project automates sales reporting using Excel VBA, reducing manual effort and improving efficiency. It includes automated data processing and a dynamic dashboard for analysis.

## 🚀 Key Features
- Automated data processing using VBA macros
- One-click report generation
- Dynamic dashboard with KPIs
- Reduced manual work and errors

## 📊 Dashboard Insights
- Sales trends and performance tracking
- KPI monitoring (Revenue, Orders, Growth)
- Automated summary reports

## 🛠 Tools & Technologies
- Microsoft Excel
- VBA (Visual Basic for Applications)
- Data Automation & Dashboarding

4. View generated reports
